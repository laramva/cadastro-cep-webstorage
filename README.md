Cadastro de Usuários com Busca de CEP

Teste o cadastro aqui: https://laramva.github.io/cadastro-cep-webstorage/

Este projeto é uma aplicação simples de cadastro que preenche automaticamente o endereço a partir do CEP utilizando a API ViaCEP.
Além disso, todos os dados digitados são salvos no Web Storage API, garantindo que o usuário não perca informações ao atualizar a página.

Funcionalidades

Preenchimento automático de endereço via Fetch API + ViaCEP

Armazenamento dos dados do formulário usando localStorage

Restauração automática dos campos ao recarregar a página

Validação básica do CEP

Layout exclusivo criado do zero, com foco em acessibilidade e boa experiência de uso

Tecnologias utilizadas

HTML5

CSS3

JavaScript (ES6+)

Fetch API

Web Storage API (localStorage)

ViaCEP API
