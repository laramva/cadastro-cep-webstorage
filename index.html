As informações são mantidas apenas no seu navegador usando Web Storage.

Como funciona

  • ViaCEP + Fetch API
    Ao digitar o CEP, a aplicação consulta a API pública ViaCEP e preenche automaticamente o endereço.
  • Web Storage API
    Cada campo digitado é salvo no navegador usando localStorage. Se você recarregar a página, os dados continuam lá.
  • Experiência contínua
    Útil para formulários longos, onde o usuário não pode correr o risco de perder as informações se a página for fechada por engano.
